Introduction

The MetaProteomeAnalyzer (MPA) constitutes a distributed computing software pipeline to analyze metaproteomics datasets. It consists of a processing server application and a graph-database driven interactive client. The software is being developed in the Java programming language and uses a neo4j graph database backend for flexible and user-defined data querying of proteomics results.

For more information, please visit the Wiki pages at the MPA Homepage

Current release version: 1.0.1

Client application

The client comes with the following key features:

  • Intuitive graphical user interface
  • Project and experiment management
  • Top-down analysis
  • Label-free quantification methods
  • Graph database-driven
  • Interactive Overview (Charts and Heatmap)
